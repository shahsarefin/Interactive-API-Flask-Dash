Image Converter & Data Generator

This is a simple web application that allows you to convert images from one format to another and generate fake data for testing purposes.

Usage Instructions

Image Conversion

Convert JPEG images to PNG or BMP formats. Upload your image and select the desired output format.

  1. Navigate to the Image Convert page.
  2. Upload your JPEG image and select the desired output format.
  3. Click the "Convert" button.

Fake Data Generation

Generate fake datasets for personal, financial, contact, employment, and online information.

  1. Go to the Data Generator page.
  2. Select the type of data and the quantity.
  3. Submit the form to generate data.
  4. Json file will be automatically downloaded through your browser.
